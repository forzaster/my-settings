## Repository Purpose

個人の設定ファイル（dotfiles）置き場。各ツールの設定ファイルをバージョン管理する汎用リポジトリ。

設定ファイルはドメイン（ツール・用途）ごとにフォルダを切って格納する。

```
my-settings/
├── tmux/       # tmux設定
├── docker/     # Docker関連
└── <tool>/     # 新しいツールはここに追加
```
